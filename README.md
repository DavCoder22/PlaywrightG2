# Playwright QA - TaskFlow

Tests de QA automation para TaskFlow (desplegada en Vercel).

## Ejecutar local

```bash
npm install
npm test
```

## Tests

- `smoke.spec.js` - Tests básicos de carga y navegación
- `auth.spec.js` - Tests de validación de formularios

## CI/CD

GitHub Actions ejecuta los tests automáticamente.

Ver: https://github.com/DavCoder22/PlaywrightG2/actions
